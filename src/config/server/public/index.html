API Documentation - Clone-Spring-MVC

Esta API proporciona endpoints para gestionar usuarios, productos y datos de prueba del proyecto Clone-Spring-MVC. Todos los endpoints están bajo el prefijo /api.

Users Controller

GET /api/users
Obtiene todos los usuarios registrados.
POST /api/users
Crea un nuevo usuario.
PUT /api/users/:id
Actualiza un usuario existente.

Products Controller

GET /api/products
Obtiene todos los productos.
POST /api/products
Crea un nuevo producto.
PUT /api/products/:id
Actualiza un producto existente.

Seed Controller

POST /seed/run
Generación de semilla de datos
GET /seed/status
Estado de los datos semilla
